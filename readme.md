# Simple HTTP GET Web Client

Web Cilente

### Requirements

[NodeJS](https://nodejs.org/en/)
[Yarn](https://yarnpkg.com/en/)


## How to run

After clone this repository perform yarn command

```
$ yarn
```

Create .env file from .env_example model, and run yarn start

```
$ yarn start
```
